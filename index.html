Introducción a la Informática

Aprende los fundamentos de la informática, incluyendo hardware, software y sistemas operativos.

Este módulo también cubre la historia de la informática y su impacto en la sociedad moderna.

Además, se exploran conceptos clave como algoritmos, estructuras de datos y redes de computadoras.

Los estudiantes obtendrán una comprensión sólida de cómo funcionan los sistemas informáticos.

Programación Web

Domina HTML, CSS y JavaScript para crear sitios web interactivos y dinámicos.

Incluye prácticas con frameworks populares como React y Bootstrap.

Aprenderás a diseñar interfaces de usuario atractivas y a implementar funcionalidades avanzadas.

Este módulo también cubre la optimización de sitios web para mejorar su rendimiento y accesibilidad.

Desarrollo de Proyectos

Aplica tus conocimientos en proyectos prácticos para consolidar tus habilidades.

Trabaja en equipo para desarrollar aplicaciones web completas y funcionales.

Se incluyen metodologías ágiles como Scrum para la gestión de proyectos.

Los estudiantes presentarán sus proyectos finales para recibir retroalimentación y mejorar.

Seguridad Informática

Descubre cómo proteger sistemas y datos frente a amenazas cibernéticas.

Aprende sobre encriptación, firewalls y mejores prácticas de seguridad.

Este módulo también aborda temas como la gestión de contraseñas y la protección de redes.

Se incluyen estudios de casos reales para comprender los desafíos de la seguridad informática.
